# [<sub><img src="Assets/Assets.xcassets/AppIcon.appiconset/128.png" width="64" height="64" /></sub>](https://github.com/leonardodino/SilkScreen/releases/latest/download/SilkScreen.zip) <sup><sup><sup>SilkScreen.app</sup></sup></sup>

#### tiny macOS app for pixel-perfect UI

spiritual successor of [**GluePrint**](https://web.archive.org/web/20161114092248if_/http://glueprintapp.com/)<sup> [RIP]</sup>


#### ✦&ensp;install:
1. download lastest version [**here**](https://github.com/leonardodino/SilkScreen/releases/latest/download/SilkScreen.zip)
2. unzip and move `SilkScreen.app` to `/Applications`

#### ✦&ensp;usage:

1. open any image file
2. drag anywhere to move the window
3. use mouse/trackpad scroll to set opacity
4. tweak your code until it matches the mockup


&emsp;&rarr;&ensp;still in doubt? [watch the original demo of GluePrint](https://vimeo.com/727110374)

<details>
<summary><b>for other actions use native macOS keybindings</b> [click to expand]</summary>

> | action | |
> |:---|---:|

> | hide window | <kbd>⌘H</kbd> |
> | miniaturize window | <kbd>⌘M</kbd> |
> | close window | <kbd>⌘W</kbd> |
> | quit app | <kbd>⌘Q</kbd> |
>
> <br>**NB**: there are no global shorcuts:<br>
> ensure SilkScreen is the active app for using the actions above
